# Guildify
Sync your Spotify listening activity to your Guilded status. Includes automatic calculation of song length to prevent ratelimiting abuse/issues.

You will need:
- Spotify developer account
- Guilded account
- a brain

Create a Spotify app, copy your client ID and secret into the config file and add a redirect to http://localhost

Rename example.json and fill in the fields. 
- Emoji takes an emoji ID
- Paste your Guilded cookie into cookie
- Paste your client ID into client
- Paste your client secret into secret

Planning to add other platforms in the future.
